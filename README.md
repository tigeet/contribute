# Contribute
## Usage
``win + r`` $\to$ ``taskschd.msc``  
Создать простую задачу 
Триггер - ``При запуске компьютера``  
Выберите действие для задачи - ``Запустить программу``  
Программа или сценарий - ``powershell``  
Добавить аргументы - ``-File <dir>/script.ps1``
